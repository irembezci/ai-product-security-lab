
# AI Product Security Lab

A hands-on portfolio repository focused on AI Product Security, threat modeling, risk assessment and security reviews of AI-powered products and agentic systems. The goal of this repository is to document a structured learning journey while building practical skills in AI Product Security through real-world case studies and security assessments.


## Objectives

This repository focuses on developing practical experience in:

* AI Product Security
* Threat Modeling
* AI Risk Assessment
* Agent Security
* OWASP Top 10 for LLM Applications
* OWASP Top 10 for Agentic Applications
* Secure AI Design
* AI Governance and Risk Management


## Methodology

Each project follows a structured security review process:

1. Product Discovery
2. Architecture Review
3. Asset Inventory
4. Trust Boundary Analysis
5. Threat Modeling
6. Risk Assessment
7. Security Requirements
8. Security Recommendations

The objective is not only to identify risks, but also to understand business context and propose practical security controls.

## Disclaimer

This repository contains independent security reviews and educational analyses of publicly available AI applications. The analyses contained in this repository represent personal research and learning activities and should not be interpreted as official security assessments.


## Learning Philosophy

The purpose of this lab is to approach AI systems from the perspective of an AI Product Security Analyst. Rather than focusing exclusively on exploitation, the emphasis is placed on understanding products, identifying risks, evaluating business impact, and recommending security improvements.
